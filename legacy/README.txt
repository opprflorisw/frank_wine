FRANK'S WINE ALMANAC
====================

An interactive, fully offline atlas of French wine.

HOW TO OPEN
  Double-click  index.html  (opens in any modern browser — Chrome, Edge, Safari, Firefox).
  No internet connection is required. Nothing is downloaded or sent anywhere.

WHAT'S INSIDE
  index.html         Interactive map of France's 13 wine regions
  regions.html       Sortable table of all regions
  houses.html        All 246 wine houses — filter by region, wine type or grape
  grapes.html        Every grape variety and who grows it
  villages.html      All 182 wine villages, geolocated
  best-regions.html  Two-axis scorecard (terroir/climate + visit experience)
  trips.html         17 curated, visitor-friendly wine-tasting road trips
  assets/            Shared styles, data and scripts (do not separate from the pages)
  exports/           A place to keep snapshots you generate (optional)

SHARING ON MOBILE / BY EMAIL
  To share a single region or a wine trip, open it and click
  "Export snapshot" / "Export this trip". That creates ONE self-contained
  HTML file (in your Downloads folder) that you can email or open on a phone.
  For a PDF, use your browser's Print > Save as PDF on any page.

NOTE
  Wine trips list only producers that genuinely receive visitors, but opening
  hours change seasonally — always confirm and book ahead before travelling.
  Region scores are an informed editorial guide, not absolute truth.
