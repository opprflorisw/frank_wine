# 🍷 Frank's Wine Almanac

An interactive atlas of French wine — now a **React + Convex** web app with an **AI sommelier** you can chat with.

Covers all 13 major wine regions, 246 celebrated wine houses (with grapes, classifications & flagship wines), 182 wine villages, a terroir/visit scorecard, and 17 curated wine-tasting road trips — all served live from a Convex database, with an AI guide that answers questions using that same data.

> This is the upgraded successor to the original standalone HTML almanac, which is preserved under [`legacy/`](./legacy).

## Stack

| Layer | Tech |
|------|------|
| Frontend | Vite + React + TypeScript + React Router |
| Backend / DB | [Convex](https://convex.dev) (database, queries, the AI action) |
| AI | Anthropic Claude (tool-calling over the wine data, runs inside a Convex action) |
| Hosting | Vercel (frontend) + Convex Cloud (backend) |

## Features

- **Interactive map** — pan/zoom SVG map of France; click a region to zoom in and reveal its villages, wine route and houses.
- **Ask the Sommelier** — a chat agent (Claude) with tools (`list_regions`, `get_region`, `search_houses`, `list_trips`, `list_grapes`) that looks up real data from the database to answer.
- **Regions / Houses / Grapes / Villages** — sortable, filterable, all live from Convex.
- **Best Regions** — two-axis terroir + visit scorecard.
- **Wine Trips** — curated, visitor-friendly itineraries with route maps.

## Local development

```bash
npm install
npx convex dev      # provisions a backend + watches functions (terminal 1)
npm run dev         # Vite dev server (terminal 2)
# seed the database (once):
npx convex run seed:run
```

Enable the AI guide locally:

```bash
npx convex env set ANTHROPIC_API_KEY sk-ant-...
# optional: choose a model (default claude-sonnet-4-6)
npx convex env set CHAT_MODEL claude-sonnet-4-6
```

## Deploying

**Convex (backend):**
```bash
npx convex login          # link to your Convex account
npx convex deploy         # creates the production deployment
npx convex run seed:run --prod        # seed production
npx convex env set ANTHROPIC_API_KEY sk-ant-... --prod
```

**Vercel (frontend):** import the GitHub repo into Vercel. `vercel.json` already sets the build command to `npx convex deploy --cmd 'npm run build'`. Add this Vercel environment variable:

- `CONVEX_DEPLOY_KEY` — a **production deploy key** from the Convex dashboard (Settings → Deploy keys). `convex deploy` uses it to deploy the backend and inject `VITE_CONVEX_URL` into the build.

Then every push to `main` deploys the Convex backend and the Vite frontend together.

## Data pipeline

The seed data (`convex/almanacSeed.json`) is generated by the Python pipeline in [`legacy/build/`](./legacy/build) (geometry from open GeoJSON, villages geocoded via geo.api.gouv.fr, per-house varietals & climate web-researched). To refresh: regenerate `almanac.json` there, copy it to `convex/almanacSeed.json`, and re-run `npx convex run seed:run`.
